Tesla Stock Price Prediction

Constructed a Sequential LSTM model for time series prediction with 50 units and ReLU activation. Incorporated a Dropout layer (0.2) to mitigate overfitting. The Dense layer produces a single output. The model, totaling 14,251 parameters, is optimized and evaluated using specified algorithms. This LSTM architecture excels in capturing sequential patterns, making it suitable for time series tasks.
